# Escena 3 — El salón / La pelea

_Con Sarah fuera del salón, Thomas ya no se contiene._

---

**THOMAS:** _(a Christian, con la voz temblando de furia)_ ¿Quién te creés que sos?

**CHRISTIAN:** _(lo enfrenta)_ El que la conoce hace tres años. ¿Y vos quién sos? ¿El que la droga?

_Thomas lo empuja. Christian responde. La pelea escala rápido, visible para todo el salón. Hay gritos, alguien tira una mesa._
